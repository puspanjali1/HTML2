Bootstrap is a component based library , it uses grid structure and it will help to make responsive website easily.

How to add bootstrap to your project
step1: goto official website 
https://getbootstrap.com/
step2: goto docs ---> introduction
step3: copy the css cdn link and paste inside head
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">


To understand Bootstrap, you need to understand following three things
1. layout
2. colors
3. media queries/ break points.

Colors: bootstrap offers by default eight colors
primary --  blue
secondary -- grey
success --- green
danger ---  red
info ---  skyblue
warning --- light gold
dark/black -- black
light/white -- white

2. layouts
Bootstrap follows grid layout with row and col as rows and columns.
Every contains 12 columns
you can specify any row as row classname and columns as col or col-values
